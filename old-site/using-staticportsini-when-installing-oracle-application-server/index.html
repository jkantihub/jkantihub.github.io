Using staticports.ini when installing Oracle Application Server

Oracle

While installing Oracle application server for a number of reasons you would want that you can control the ports configuration. The best way to do this on a new install is to use staticports.ini file to specify the ports.

A sample staticports.ini is shown below.

## staticports.ini - default # J2EE and HTTP Server Oracle HTTP Server port = 7777 Oracle HTTP Server Listen port = 7777 Oracle HTTP Server SSL port = 4443 Oracle HTTP Server Listen (SSL) port = 4443 Oracle HTTP Server Diagnostic port = 7200 Java Object Cache port = 7000 DCM Discovery port = 7100 Oracle Notification Server Request port = 6003 Oracle Notification Server Local port = 6101 Oracle Notification Server Remote port = 6200 Application Server Control port = 18100 Application Server Control RMI port = 18140 Oracle Management Agent port = 18120 Log Loader port = 44000 ASG port = 7890 # Infrastructure Oracle Internet Directory port = 1389 Oracle Internet Directory (SSL) port = 1636 Enterprise Manager Console HTTP Port (orcl) = 5500 Enterprise Manager Agent Port (orcl) = 1830
