Oracle supports mysql client

General

With the completion of the BEA strategy we now have Weblogic being promoted as the core J2EE engine. Added to that is the point that Weblogic 10.0 bundles mysql 5.0 support right out of the box.

Go look http://edocs.bea.com/wls/docs100/notes/new.html

The real point is

**MySQL 5.0 Support** WebLogic Server bundles a driver for MySQL 5.0.x and supports MySQL 5.0.x. See Supported Database Configurations.

So no matter what way which way you go, the end of the road is pretty well clear.

Free Software is really changing the way.
