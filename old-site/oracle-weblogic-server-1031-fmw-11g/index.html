Weblogic 10.3.1 install review for FMW 11g

Weblogic is the base java container for FMW 11g.The first piece of software you will put for any FMW 11g install.Oracle Weblogic Server is a collection of the application server,admin console and other components.

What I like about the install is the number of options you have.The install can be done online using the net installer or using the downloaded software.You have a text as well as gui based installer.You can also script the install for silent mode.

The install creates a middleware home for you.This will be the top level directory under which additional FMW 11g components would be installed.In weblogic , each OC4J container is now a domain.As you install the various components eg. Forms,Reports etc new domains get created.

There were 108 different ways you could have Oracle Application Server 10g put into use.With Fusion Middleware 11g I can see much more.
