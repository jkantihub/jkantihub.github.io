PHP Quercus with OC4J ( JDeveloper 11 preview )

General

PHP has eveloved to be a mature web scripting language.
Had used PHP in my earlier incarnation :) , for those who know.

Recently i was looking at some blogs and heard that running php on java is picking up some interest.
The Quercus team at http://quercus.caucho.com/ has really done some interesting stuff.

Now what i wanted to do was to see if its possible to run Quercus on Jdev.
I downloaded the latest jdev preview from http://www.oracle.com/technology/products/jdev/index.html and configured it with JDK 1.5.10.
The next step was to get the quercus ( http://quercus.caucho.com/download/quercus-3.1.1.war ).Downloaded the beta , just to drive.

Imported the WAR file into my JDev project and ran it in the embedded OC4J.
And viola i had php running 100% java.
http://127.0.0.1:8988/Application1-Project1-context-root/index.php
