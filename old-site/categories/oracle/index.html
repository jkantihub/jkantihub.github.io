Netbackup and oracle_link

General Oracle

Whenever you need to configure the Oracle database with netbackup you need to set up a link from the netbackup client to ORACLE_HOME. The simplest way is to go to netbackup/bin and execute the “oracle_link” script. The script links libobk.a64 to $ORACLE_HOME/lib/libobk.a64. The common error here is “ORA-27211: Failed to load Media Management Library”

Pointing the Oracle inventory to proper ORACLE_HOME after cloning

Oracle

Once the clone of a database is done it is required that it be associated with the proper inventory. There are a number of ways one can do this but the best according to me is to use runInstaller itself. The procedure to do this is to use the attachhome option of runInstaller from the $ORACLE_HOME/oui/bin directory ./runInstaller -silent -attachHome ORACLE_HOME="/app/oracle/database/10.2.0" ORACLE_HOME_NAME="databasename" Starting Oracle Universal Installer… No pre-requisite checks found in oraparam.ini, no system pre-requisite checks will be executed.

Using staticports.ini when installing Oracle Application Server

Oracle

While installing Oracle application server for a number of reasons you would want that you can control the ports configuration. The best way to do this on a new install is to use staticports.ini file to specify the ports. A sample staticports.ini is shown below. ## staticports.ini - default # J2EE and HTTP Server Oracle HTTP Server port = 7777 Oracle HTTP Server Listen port = 7777 Oracle HTTP Server SSL port = 4443 Oracle HTTP Server Listen (SSL) port = 4443 Oracle HTTP Server Diagnostic port = 7200 Java Object Cache port = 7000 DCM Discovery port = 7100 Oracle Notification Server Request port = 6003 Oracle Notification Server Local port = 6101 Oracle Notification Server Remote port = 6200 Application Server Control port = 18100 Application Server Control RMI port = 18140 Oracle Management Agent port = 18120 Log Loader port = 44000 ASG port = 7890 # Infrastructure Oracle Internet Directory port = 1389 Oracle Internet Directory (SSL) port = 1636 Enterprise Manager Console HTTP Port (orcl) = 5500 Enterprise Manager Agent Port (orcl) = 1830 Feel free to post comments with your feedback.
