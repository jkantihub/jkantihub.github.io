Netbackup and oracle_link

General Oracle

Whenever you need to configure the Oracle database with netbackup you need to set up a link from the netbackup client to ORACLE_HOME. The simplest way is to go to netbackup/bin and execute the “oracle_link” script. The script links libobk.a64 to $ORACLE_HOME/lib/libobk.a64. The common error here is “ORA-27211: Failed to load Media Management Library”

Pointing the Oracle inventory to proper ORACLE_HOME after cloning

Oracle

Once the clone of a database is done it is required that it be associated with the proper inventory. There are a number of ways one can do this but the best according to me is to use runInstaller itself. The procedure to do this is to use the attachhome option of runInstaller from the $ORACLE_HOME/oui/bin directory ./runInstaller -silent -attachHome ORACLE_HOME="/app/oracle/database/10.2.0" ORACLE_HOME_NAME="databasename" Starting Oracle Universal Installer… No pre-requisite checks found in oraparam.ini, no system pre-requisite checks will be executed.
