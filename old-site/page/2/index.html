Fusion Middleware 11g , another look at the architecture.

Oracle

Oracle released Fusion Middleware 11g recently.This release has a lot of changes. To enumerate a few , weblogic is now the core java container.Apart from this ,11g identity management uses Access Manager as the authorization broker.There are more such subtle gems hidden inside the release. The traditional components like Forms,Reports, Portal and Webcache continue to make use of SSO and OIDDAS.These components need a lot more detailed planning before you even start thinking of installation.So , if you have Ebusiness Suite added in the gamit , your options are a bit limited.
